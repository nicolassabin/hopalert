WebCodex c'est quoi?

La page WebCodex est un outil informatique en marque blanche permettant la mise en relation entre deux personnes, l'une ayant égarée un objet et l'autre ayant trouvé cet objet.

WebCodex inclut et gère le traitement des données enregistrées en respectant scrupuleusement l'anonymat. WebCodex est un éditeur de codes uniques.

L'usage de cette page est sans engagement, sans publicité et sans spam.

Comment ça marche?

Vous avez reçu ou acheté un objet estampillé d'un code unique WebCodex. Il vous suffit d'enregistrer ce code objet d'un clic sur 'j'enregistre mon code' et d'y préciser vos coordonnées de contacts. Ces mentions seront totalement cryptées.

Vous égarez un de vos objets; la personne qui trouvera cet objet saura vous contacter en indiquant le numéro de votre code objet sur la page WebCodex par un clic sur 'j'ai trouvé un objet'.

Vos coordonnées d'enregistrement ne seront jamais visibles. La personne ayant trouvé votre objet vous proposera un moyen d'entrer en relation directe afin d'être en mesure de vous restituer votre bien. Dans la relation, le code WebCodex de votre objet devient votre code ADN!
